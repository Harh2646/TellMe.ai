# TellMe.ai

**AI-Powered Multi-Document Reader & Voice Assistant**

TellMe.ai is a lightweight AI system that can read multiple PDF documents, answer user questions based on their content, and provide voice-based responses. The system is designed to run efficiently on standard laptop hardware without requiring cloud services or GPUs.

## Key Features
- Upload and process multiple PDF documents
- Document-based question answering
- Chat-style conversational interface
- Adaptive document summarization
- Voice-based response output

## Project Motivation
Reading and extracting information from large documents such as research papers, legal contracts, and financial reports is time-consuming. TellMe.ai aims to simplify this process using AI-driven document understanding.

## Technology Overview
- Python
- Lightweight NLP models
- Local CPU-based execution
